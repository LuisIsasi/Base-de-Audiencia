# AudienceDB

## Local Development

```
# Setup local environment (via Docker Compose)
make init
make app-init

# Start server (access at localhost:7979)
make app-start

# Stop server
make app-stop

# Run tests
make test-run

# Open command-line shell
make app-shell

# Rebuild container (same as app-init)
make app-build
```
